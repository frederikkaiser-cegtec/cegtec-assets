# CegTec Assets

Static asset hosting for CegTec – images served via `raw.githubusercontent.com`.

## Ordnerstruktur

| Ordner | Inhalt |
|--------|--------|
| `linkedin/` | LinkedIn Post Infografiken |
| `case-studies/` | Case Study Grafiken |
| `academy/` | CegTec Academy Assets |
| `brand/` | Logos, Icons, Brand Assets |
| `misc/` | Sonstiges |

## URL-Schema

```
https://raw.githubusercontent.com/frederikkaiser-cegtec/cegtec-assets/main/{ordner}/{dateiname}
```

### Beispiel

```
https://raw.githubusercontent.com/frederikkaiser-cegtec/cegtec-assets/main/linkedin/linkedin-01-seo-hidden-cost.png
```

Einfach die URL in Notion, Webflow oder andere Tools als Bild-URL einbetten.
